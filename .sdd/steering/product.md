# Product Overview

A full-stack AI chatbot web application that provides real-time streaming conversations powered by Claude.

## Core Capabilities

- **Streaming chat**: Real-time token-by-token response delivery via ReadableStream
- **Markdown rendering**: Rich assistant responses with prose-styled markdown
- **Conversation context**: Full message history sent with each request for multi-turn dialogue
- **Input validation**: Character limits with truncation feedback

## Target Use Cases

- Interactive Q&A with Claude as the backend LLM
- Proof-of-concept for Anthropic SDK integration in Next.js applications
